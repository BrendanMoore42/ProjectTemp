Color to Py

Ask what colour you want to search for in an image or window and itll grab the hex codes and name for each shade of color found.

or input a painting or whatever and itll list all of the shades and names of the colours that are tied to names

>>python main.py

>>input file

>>read rbg values (either: each cell, or the highest collections of rgb values that == a color.)

>>link rgb sequences to bins that associate with hex codes

>>return value to hexcode key
